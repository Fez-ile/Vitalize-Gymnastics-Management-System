The **Vitalize Gymnastics Management System** is a PHP-based web application designed to help gymnastics coaches manage training programs and allow gymnasts to enrol, track attendance, and monitor progress over time.

This system was developed as part of an academic project and demonstrates core web development concepts such as CRUD operations, database integration, and progress tracking using PHP and MySQL.


## Features

### Program Management (Coach)
- Add new gymnastics programs
- Edit existing program details
- Delete outdated or cancelled programs
- View all programs in a structured table format

### Gymnast Enrolment
- Gymnasts can enrol in available programs
- Capture gymnast details (name, age, experience level)
- Link gymnasts to selected programs

### Attendance & Progress Tracking
- Mark gymnast attendance per session
- Add progress notes or scorecards
- Display progress using percentage indicators
